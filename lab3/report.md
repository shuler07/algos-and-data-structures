## Лабораторная работа №3
##### Цель работы:
Составить программу выполнения заданных преобразований арифметических выражений с применением деревьев, реализовав преобразование выражения в дерево одним из известных методов и набор подпрограмм для обработки.
##### Идея, метод, алгоритм решения задачи:
Программа реализует преобразование арифметических выражений с использованием деревьев для варианта 18 — вынесение унарных минусов из произведений. Алгоритм работает в несколько этапов: сначала исходное инфиксное выражение преобразуется в постфиксную запись алгоритмом Дейкстры (с обработкой унарных минусов как отдельного оператора '~'). Затем по постфиксной записи рекурсивно строится бинарное дерево, где операторы становятся внутренними узлами, а операнды — листьями. После построения выполняется обход дерева с преобразованием: обнаруженные унарные минусы (узлы '~') в произведениях выносятся наружу согласно правилам варианта 18. Результат преобразования выводится в виде дерева и в текстовом представлении.
##### Сценарий выполнения работы:
Работа началась с реализации структур данных для стека и дерева, а также базовых операций над ними. Затем разработан модуль преобразования инфиксной записи в постфиксную с корректной обработкой унарных минусов. Далее реализовано построение дерева из постфиксной записи и обратное преобразование дерева в строку. Основная функция преобразования (вынесение унарных минусов) реализована рекурсивным обходом дерева с модификацией узлов. Для проверки корректности созданы модульные тесты на нескольких выражениях, включая граничные случаи (один унарный минус, несколько унарных минусов в разных частях выражения). Программа организована в виде циклического меню, позволяющего вводить выражения и получать результат преобразования.
