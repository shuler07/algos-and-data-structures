## Лабораторная работа №2
##### Тема:
Сортировка и поиск
##### Цель работы:
Составить программу на языке Си с использованием процедур и функций для сортировки таблицы 
заданным методом и двоичного поиска по ключу в таблице.
##### Идея, метод, алгоритм решения задачи:
Программа реализует сортировку таблицы методом линейного выбора с подсчетом для комбинированного типа ключа (строка + целое) с хранением данных и ключей вместе. Основная идея алгоритма — построение вспомогательного массива счетчиков, где для каждого элемента подсчитывается количество элементов, которые меньше или равны ему (с учетом приоритета: сначала сравнение по целочисленной части ключа, затем по строковой). После подсчета выполняется перестановка элементов в соответствии с полученными позициями. Для сравнения ключей реализована функция compare_table_data, учитывающая комбинированную структуру: сначала сравниваются числа, при их равенстве — строки.
##### Сценарий выполнения работы:
Работа началась с определения структуры таблицы TableData, включающей целочисленную и строковую части ключа, а также строковое значение. Реализованы функции ввода-вывода read_table и write_table для работы с текстовыми файлами. Основной алгоритм сортировки реализован в linear_counting_sort: сначала заполняется массив счетчиков попарным сравнением всех элементов, затем элементы переставляются на свои позиции согласно счетчикам. Для тестирования создан модуль test-lab2.c, проверяющий корректность ввода-вывода и сортировки на случайных данных. Основная программа lab2.c последовательно демонстрирует сортировку для трех случаев: упорядоченного, обратного и случайного порядка элементов (с использованием генерации псевдослучайных чисел), выводя исходное и отсортированное состояния таблицы.
