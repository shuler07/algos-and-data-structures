## Лабораторная работа №1
##### Тема:
Динамические структуры данных. Обработка деревьев
##### Цель работы:
Составить программу на языке Си для построения и обработки дерева общего вида с реализацией операций добавления, удаления, визуализации и подсчёта числа вершин.
##### Идея, метод, алгоритм решения задачи:
Дерево общего вида реализовано через узлы с фиксированным массивом указателей на потомков (до 5, число можно менять). Добавление узла выполняется рекурсивным поиском родителя по значению с последующим добавлением в конец массива потомков. Удаление узла рекурсивно удаляет всё поддерево с коррекцией массива потомков родителя. Визуализация использует рекурсивный обход с выводом отступов. Подсчёт числа вершин реализован рекурсивным обходом с суммированием 1 для каждого узла. Интерфейс — текстовое меню.
##### Сценарий выполнения работы:
Сначала разработана структура узла и базовые функции создания и добавления. Затем реализовано рекурсивное удаление поддерева с освобождением памяти. Функция подсчёта вершин реализована через рекурсивный обход с накоплением суммы. После отладки создано меню и модульные тесты для проверки корректности работы всех операций.
